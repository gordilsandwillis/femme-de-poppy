== Description ==

Allows you to change the checkboxes in the taxonomy metabox to Radio buttons, Checkbox or Checkbox with Select All possibility.

You can switch between displaying taxonomies, even custom taxonomies in different ways.

The "select all" functionality can be enabled only if the taxonomy display type is checkbox.

== Installation ==
1. Upload the plugin files to the plugins directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Use the Settings->Moove taxonomy buttons screen to configure the plugin
